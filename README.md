# FE_2W_YoutubeClone

배포 URL (작업중)

### 페이지 구성
- Home: 검색창과 동영상 카드 목록을 보여주는 페이지
- Video: 동영상 재생하는 페이지
- VideoDetail: 동영상 상세정보 페이지

### ❔ 선택

- [ ] 유튜브 API를 활용한 검색 및 동영상 재생 기능 구현
- [ ] ailwind CSS를 이용한 디자인 구현
- [ ] React Router를 이용한 페이지 이동 구현
- [ ] React Query를 이용한 데이터 요청 및 관리 구현
- [ ] ocket을 이용한 채팅 구현
